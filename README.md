# Islamic Login Form

A modern **Islamic style login form** built using **HTML and CSS** with a glassmorphism design.

## ✨ Features

* Modern glassmorphism UI design
* Responsive layout
* Social login icons
* Clean and simple code
* Google Fonts integration
* Font Awesome icons

## 🛠️ Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome

## 📂 Project Structure

```
project-folder
│
├── index.html
├── style.css
└── background.jpg
```

## 🚀 How to Use

1. Download or clone the repository
2. Open `index.html` in your browser
